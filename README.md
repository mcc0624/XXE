# 二、靶场搭建

现将代码下载到本地
```
git clone https://github.com/mcc0624/XXE.git
```

进入目录并使用docker-compose 进行部署

```
cd XXE
docker-compose up -d 
```

部署完成后访问主机80端口即可看到靶场内容
